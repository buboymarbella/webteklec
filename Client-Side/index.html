CSS History and Origin

CSS or cascading style sheetsspan> is used for describing the presentation of an html document. Though CSS is not limited to only HTML documents it may also be applied to SVG, XML and XHTML docs.

CSS was proposed by Hakon Wium Lie at around that time several proposals for style sheet languages was proposed. This led to the World Wide Web Consortium's first CSS Recommendation. As his proposal was influential Bert Bos became co-author of CSS1


CSS Variations

  • CSS 1 - includes font properties, line spacing and text attributes, text colors, backgrounds and positioning options such as margin and paddings.
  • CSS 2 - added capabilities for positioning of elements such as relative, absolutes and Z - index. Media type concepts and font properties.
  • CSS 2.1 - This first revision of the CSS 2 moves forward by fixing errors and removing not fully supported features from the previous version.
  • CSS 3.0 - Preserves backwards compatibility. Now divided into modules.

CSS Selectors

In CSS in order to refer to an element that we want to style we need to indicate a selector. This can be done in different ways with different selectors.

Selector Syntax Description
type p Selects a specific element to style
class .proj Selects an element with a matching class attribute
id #proj Selects an element with a matching id attribute
* * Selects all elements

CSS Combinators

To further specify the element you want to style combinators offer a lot of variations in selecting an element.

Combinators Syntax Description
Descendant Selector A B This combinator selects children of the first specified element even if not direct descendants
Child Selector A > B The combinator selects children that are direct descendants of the first specified element
General Sibling Selector A ~ B This combinator selects the latter element that follows after the former element not necessarily immediately
Adjacent Sibling Selectors A + B This combinator selects the latter element that follows immediately after the former element

CSS Psuedo Elements

A Psuedo element allows the user to style specific part of a selected element.Some of them are:

  • ::after - This will be placed after the element.
  • ::before - This will be placed before the element.
  • ::first-letter - Only applies to the first letter of the element.
  • ::first-line - Only applies to the first line of the element.
  • ::selection - Applies to a portion of a document highlighted by the user.
